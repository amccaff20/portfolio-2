Hiking Trail Finder

Find nearby trails and weather information

GitHub Repo. Live App. I created this hiking trail finder to provide a clean and simple way for users to quickly gather hiking trails options in their area. There are some basic filters that allow the user to declare a maximum number of results, the minimum length of the hike, and how to sort the hikes. In addition to this, current and future weather conditions are provided so the user can plan accordingly. The final feature is the map, where users can click on the coordinates for a certain trail and get a precise marker on the map for reference.

  • Tech: HTML, CSS, JavaScript, REST API
